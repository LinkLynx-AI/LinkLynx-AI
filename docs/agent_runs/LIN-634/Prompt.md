# LIN-634 Prompt

## Goal
- チャンネルの親子関係（カテゴリ配下、スレッド）をDBで表現可能にする。
- 既存 `channels(guild_text/dm)` との互換を維持したまま additive に導入する。
- 並び順とアーカイブ状態を保持する最小属性を追加する。

## Non-goals
- スレッド参加者管理の実装。
- UI表示順ロジックの最終仕様実装。
- ScyllaメッセージSoR構造の変更。

## Done conditions
- カテゴリ配下に複数チャンネルを紐づけられる。
- 親メッセージに紐づくスレッドチャンネルを識別できる。
- 既存 guild_text / dm データへの破壊的変更がない。
